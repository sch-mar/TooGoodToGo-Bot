## config

create a ```.config``` file using the following template

```ini
[telegram]
api_key = <your bots api key>

[tgtg]
mail = <the mail of your tgtg account>
```

## menu

### /start

- welcome + info

### /setup

- new user
- get credentials

### /setinterval

- change default interval

### /deleteuser

- delete all user information
